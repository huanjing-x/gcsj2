# 代码片段管理系统 - 使用说明书

---

## 一、系统简介

代码片段管理系统是一款基于 Web 的云端代码管理工具。支持代码片段的分类存储、标签标记、在线编辑预览、多条件检索、AI 辅助优化等功能，帮助程序员高效管理日常积累的代码。

### 核心功能

- 🔐 用户注册登录、JWT 鉴权、数据隔离
- 📁 自定义分类管理代码片段
- 🏷️ 多标签绑定，灵活筛选检索
- 📝 Monaco 在线编辑器，语法高亮、主题切换
- 🤖 AI 辅助代码优化（添加注释、BUG检测、代码重构）
- ⭐ 收藏代码、一键复制、导出文件
- 📱 移动端自适应，手机浏览器可访问

---

## 二、快速开始

### 第一步：注册账号

1. 打开系统 → 点击「立即注册」
2. 填写账号（6-20位）、密码（6-16位）、昵称
3. 点击注册 → 自动跳转登录页

### 第二步：登录

1. 输入账号和密码
2. 勾选「记住我」可延长登录有效期
3. 点击登录 → 进入系统首页

### 第三步：创建分类

1. 点击侧边栏「分类管理」
2. 点击「新增分类」→ 填写分类名称和排序号
3. 保存后在新增代码时可选择该分类

### 第四步：创建标签

1. 点击侧边栏「标签管理」
2. 点击「新增标签」→ 填写标签名称
3. 标签用于给代码片段打标记，方便筛选

### 第五步：新增代码片段

1. 点击侧边栏「新增代码」
2. 填写标题，选择编程语言
3. 在编辑器中编写或粘贴代码
4. 选择分类和标签（可选）
5. 点击「保存」

## 三、功能详解

### 3.1 代码列表

- **搜索栏**：支持按标题模糊搜索、语言筛选、分类筛选、标签筛选、仅看收藏
- **表格**：展示标题、语言、分类、标签、创建时间、收藏状态
- **操作按钮**：
  - **查看**：弹窗以只读模式预览完整代码
  - **编辑**：跳转编辑页面
  - **复制**：一键复制代码到剪贴板
  - **导出**：下载为对应后缀文件（.py/.java/.js等）
  - **收藏/取消收藏**：标记收藏状态
  - **删除**：删除代码片段（不可恢复）

### 3.2 代码编辑器

- **语法高亮**：支持 Java、Python、JavaScript、Go、SQL 等 15+ 语言
- **主题切换**：深色/浅色主题一键切换
- **自动缩进**、**行号显示**、**代码提示**、**格式化粘贴**

### 3.3 AI 代码辅助（加分功能）

在新增或编辑代码页面，选中编程语言后可使用 AI 辅助：

1. **添加注释**：AI 自动为代码添加详细中文注释
2. **BUG 检测**：AI 检查代码中的潜在问题并给出修复方案
3. **重构优化**：AI 精简代码、优化结构，提高可读性

处理完成后会弹窗询问是否覆盖编辑器内容。

> 注意：AI 功能需在配置文件中设置正确的 API 接口地址和密钥。

### 3.4 个人中心

- **修改昵称**：更改显示名称
- **修改密码**：需输入原密码验证

### 3.5 退出登录

点击右上角「退出」按钮，确认后清除登录状态并跳转登录页。

## 四、快捷键

| 操作 | 键位 |
|------|------|
| 搜索 | 在搜索框按 Enter |
| 复制代码 | 在编辑器内 Ctrl+C |
| 编辑器中保存 | 快捷键取决于编辑器 |

---

## 五、移动端使用

系统支持移动端浏览器自适应布局：
- 侧边栏自动折叠
- 表格横向滚动
- 表单单列排列

在手机浏览器中访问部署地址即可使用。
